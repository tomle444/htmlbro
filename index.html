Welcome to HTMLBro

HTMLBro is a Jekyll theme that is built using the Bootstrap Framework. This is a customizable theme that can be made to be personalized for your needs.

What is Jekyll?

Jekyll is a blog-aware, static site generator. The great thing about serving up a good ol' web page in a series of .html files is because it's 100% secure. No databases or account logins and passwords to worry about.

Combine Jekyll with Github Pages and you can have a quick and easy website for just the cost of a domain name, which is relatively cheap these days.

Liquid coding

Jekyll uses Liquid code to create these static sites. You can store data within _config.yml files and use Liquid code to loop through values.
